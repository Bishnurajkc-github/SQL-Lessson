

My suggestion for your learning order

Since you're learning Flask alongside SQL, I'd recommend this sequence after Lesson 21:

Primary Key
Foreign Key
One-to-Many Relationships
Constraints (NOT NULL, UNIQUE, CHECK, DEFAULT)
INNER JOIN
LEFT JOIN
IN, BETWEEN, CASE, AS
Subqueries
Indexes
Views
Transactions
Practice with mini-projects
SQLite with Python
SQLAlchemy
Flask database relationships
